# Calorie Counter

Calorie Counter is a web application that helps to control number of calories consumed.

## Demo

Here is a working live demo: https://www.calorie-counter.pl


![App screenshot](./app-screenshot.png?raw=true)

## Built with

- Angular 5
- Node.js/Express
- MongoDB
